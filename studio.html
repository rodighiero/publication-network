Studio

Drop in a set of text files and watch them arrange into a similarity network — embedding, similarity, and layout all run live in your browser. Your files never leave this machine.

Each file becomes a node, placed near the files it most resembles. Drop .txt or .md files (20–30 short ones works well).

Each node links to its single closest file, but only if their similarity clears this value. Raise it and the network fragments; lower it and everything connects.
